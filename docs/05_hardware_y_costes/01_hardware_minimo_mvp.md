# Hardware mínimo para MVP en casa

## Imprescindible
- ordenador personal actual,
- 1 webcam 1080p,
- soporte o trípode,
- espacio en disco suficiente.

## Muy recomendable
- buena iluminación estable,
- mesa bien encuadrada,
- fondo controlado,
- red local fiable si la cámara no es USB.

## Objetivo del hardware mínimo
No maximizar potencia, sino validar:
- detección,
- conteo,
- ocupación,
- tiempos,
- eventos.

## Regla
No comprar edge hardware dedicado hasta demostrar que el pipeline funciona.
