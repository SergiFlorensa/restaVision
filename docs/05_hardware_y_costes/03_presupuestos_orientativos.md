# Presupuestos orientativos

## Perfil A — MVP austero
Objetivo:
- empezar ya con mínima inversión.

Incluye:
- webcam básica,
- soporte,
- usar el PC actual.

Rango orientativo:
- bajo.

## Perfil B — MVP serio
Objetivo:
- trabajar cómodo y dejar base para piloto.

Incluye:
- webcam mejor,
- mejor iluminación,
- quizá SSD adicional,
- quizá mini PC económico.

Rango orientativo:
- medio.

## Perfil C — Piloto edge
Objetivo:
- acercarse a escenario real.

Incluye:
- 2–4 cámaras,
- edge compute dedicado,
- monitorización,
- audio opcional.

Rango orientativo:
- medio/alto para proyecto personal, pero aún contenido comparado con soluciones empresariales.

## Política de gasto
No pagar por software mientras no haya evidencia de cuello de botella o necesidad clara.
