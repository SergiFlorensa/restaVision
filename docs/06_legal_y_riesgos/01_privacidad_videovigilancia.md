# Privacidad, videovigilancia y protección de datos

## Punto de partida
Este proyecto trata imágenes de personas, por lo que entra en una zona sensible de protección de datos.

## Referencias clave
- Guía AEPD sobre videocámaras: https://www.aepd.es/guias/guia-videovigilancia.pdf
- Ficha AEPD para establecimientos públicos: https://www.aepd.es/documento/fichas-videovigilancia-5-camaras-establecimientos-publicos.pdf
- Directrices EDPB 3/2019 sobre vídeo: https://www.edpb.europa.eu/our-work-tools/our-documents/guidelines/guidelines-32019-processing-personal-data-through-video_en

## Principios a aplicar
- finalidad clara,
- minimización,
- proporcionalidad,
- información visible,
- seguridad del tratamiento,
- limitación de conservación,
- control de acceso.

## Recomendaciones para el diseño
- inferencia local siempre que sea posible,
- no usar reconocimiento facial,
- no identificar individuos,
- evitar atributos sensibles,
- evitar “emotion AI” o inferencias invasivas,
- almacenar solo lo necesario.

## Regla para el TFG
Presentar el sistema como soporte operacional y analítica de sala, no como herramienta de vigilancia invasiva o control personal.
