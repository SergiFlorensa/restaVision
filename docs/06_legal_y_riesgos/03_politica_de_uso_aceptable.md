# Política de uso aceptable del sistema

## El sistema sí puede
- estimar estados de mesa,
- detectar cambios operativos,
- calcular tiempos,
- generar alertas de riesgo,
- apoyar decisiones humanas.

## El sistema no debe
- identificar personas concretas,
- reconocer rostros para seguimiento nominativo,
- inferir emociones como criterio principal,
- etiquetar a clientes de forma sensible,
- actuar como juez automático de fraude o impago.

## Principio operativo
La decisión final siempre es humana.
