# Services

Servicios de dominio y pipelines desacoplados:
- `vision/` para percepción visual.
- `events/` para transformación de observaciones en eventos.
- `prediction/` para ETA y modelos explicables.
- `alerts/` para reglas, scores y notificaciones.
- `maria/` para parser de instrucciones y orquestación de análisis multimodal bajo demanda.
